Karat

An Actually Minimal CSS Boilerplate

About

Karat is a responsive, minimal CSS Framework. Unlike other frameworks, like Bootstrap and Foundation, Karat gives only the bare essentials. It even better than other "minimal frameworks" that include UI that you never use, and only sit in your <head> tag slowing down your website.

Containers

Containers let you add padding to the sides of a website, and make it look better. Karat's container is responsive and gets smaller when the page is smaller. You can also add this class to <body> or other tags.

<div class="container">

Padding Element

Add a padding between elements by putting this in a div. This rule can also break text.

padding-0 0px

0px padding

padding-5 5px

5px padding

padding-10 10px

10px padding


<div class="padding-0"></div> <!-- 0px padding -->

<div class="padding-5"></div> <!-- 5px padding -->

<div class="padding-10"></div> <!-- 10px padding -->
